# 🌍 Global CO2 Emissions Dashboard

## 📊 Objetivo
Analisar emissões globais de CO2 utilizando dados reais para identificar padrões e tendências.

## 🛠️ Tecnologias
- Python
- Pandas
- Plotly
- Streamlit

## 🌐 Fonte de dados
Our World in Data

## 🔍 Insights
- China e EUA lideram emissões absolutas
- Emissões per capita variam significativamente
- Países emergentes apresentam crescimento acelerado

## 🚀 Como rodar
streamlit run app.py
